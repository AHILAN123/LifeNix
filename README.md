# LifeNix

LifeNix is an AI-powered healthcare platform that provides users with basic health insights, medical document analysis, and virtual consultation support. The system is designed to improve accessibility to healthcare services through a unified digital interface.

## Features

- Symptom-based health insights  
- Medical document summarization  
- OCR-based report analysis  
- AI-assisted health chatbot  
- Virtual consultation support  

## Technology Stack

Frontend: React, Next.js, Tailwind CSS  
Backend: Node.js, Express  
Database: MongoDB  
AI: Genkit, Google Gemini API
